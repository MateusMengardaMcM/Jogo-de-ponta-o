
# JOGO DE PONTUAÇÃO

Projeto voltado para fins educacionais, com intenção de aprender javascript, focando nos conceitos do loop FOR.


## Aprendizados

Compreendi os conceitos da condicional IF no javascript.
## Funcionalidades

- A condicional IF decide se você ganhou ou nao baseado em pontos.

## Stacks utilizada

**Front-end:** Html & JavaScript.



## Deploy

Para fazer o deploy desse projeto faça um clone do repositório e execute o arquivo em index.html.

```bash
  git clone https://github.com/MateusMengardaMcM/Jogo-de-ponta-o
```
